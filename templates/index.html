Commands


DM Commands: The below commands are handled through sending a DM to {{ bot_name_twitter }} on Twitter. You can click the link next to the command to send a DM to the Twitter bot.

  • !help: The tip bot will respond to your DM with a list of commands and their functions. If you forget something, use this to get a hint of how to do it!
  • !register: The tip bot will create an account, or validate the account that is currently tied to your Twitter. This will return a message containing your account number.
  • !balance: Returns the current balance of your account.
  • !account: Returns the account number tied to your Twitter user name.
  • !withdraw: Withdraws the balance of your tip account to a specified Vericoin account. Example: !withdraw {{ bot_account }} will withdraw to account {{ bot_account }}
  • !donate: Donates a specified amount from your tip account to the developer. Example: !donate 1 will donate 1 VRC to Developers. Thanks for your generosity!
  • !setlanguage: Change the default language of the bot. A list of available languages is provided with the !languages command. Proper use is "!setlanguage Russian" to change your language to Russian.
  • !languages: Display a list of available languages for translation.

Tweet Message Commands: Tips are sent through public tweets, retweets. These are processed in real time.

  • {{ tip_command }}: Send a tip of a specified amount to a user on Twitter.
    • Twitter Example: {{ bot_name_twitter }} {{ tip_command }} 5.01 @mderasse will send a 5.01 VRC tip to user @mderasse.
